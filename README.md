# :rocket: Speedh0ck
A linux speedhack GUI program to speed up / slow down programs<br><br>

# :wrench: Build & Run
```
git clone https://github.com/n0nexist/Speedh0ck
cd Speedh0ck
bash buildandrun.sh
```
